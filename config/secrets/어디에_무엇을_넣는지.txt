TASS 시크릿 저장 위치 안내
==========================

[실제 값 작성 파일 — 여기만 편집]
  경로: TASS/.env  (프로젝트 루트, 이 폴더의 한 단계 위)

[템플릿 — 값 넣지 않음, Git 공유용]
  경로: TASS/.env.example

[코드가 읽는 설정]
  경로: TASS/config/app_settings.py
  → 프로젝트 루트의 .env 만 자동 로드

[엔진 파라미터 — API 키 금지]
  경로: TASS/config/settings.yaml


작성 순서 (필요할 때만)
----------------------
1. LOG_LEVEL, TIMEZONE          → 지금 바로 (비밀이 아님)
2. DATABASE_URL                 → DB 연결할 때
3. JWT_SECRET, API_KEY          → REST API 붙일 때 (직접 랜덤 생성)
4. REDIS_URL                    → 캐시·운영 배포할 때
5. KIS_APP_KEY, KIS_APP_SECRET  → KIS 시세 어댑터 쓸 때


보안 주의
---------
- .env 는 Git 에 커밋하지 않음 (.gitignore 등록됨)
- oho_bujaJJ .env 와 TASS .env 는 분리 유지
- 코드·YAML·README 에 키 하드코딩 금지
- 이 txt 파일에도 실제 키를 적지 않음
