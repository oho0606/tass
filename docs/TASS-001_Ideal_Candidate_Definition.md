# 1. 목적 (Purpose)

본 문서는 TASS가 어떤 종목을 추천해야 하는지 정의하는 프로젝트의 핵심 문서이다.

이 문서는 이후 개발되는 모든 Rule, Scoring Engine, Gate System, Confidence Engine 및 AI 추천 시스템의 기준이 된다.

TASS는 모든 종목을 분석하는 것이 목적이 아니다.

**"확률적으로 가장 유리한 매수 후보를 자동으로 추천하는 것"** 이 목적이다.

# 6. v1.0에서 목표로 하는 패턴

TASS v1.0은 모든 차트 패턴을 찾지 않는다.

다음 하나의 패턴만 완벽하게 찾는 것을 목표로 한다.

## Pattern B

### 상승 추세 → 눌림목 → 재상승

이 패턴을 선택한 이유

* 스윙 트레이딩에 적합
* 손절이 명확
* 기대수익이 높음
* 기술적 분석으로 정량화하기 쉬움
* 백테스트 검증이 용이

# 10. 사용자 경험

사용자는 차트를 분석하기 위해 TASS를 사용하는 것이 아니다.

사용자는

오늘 매수할 만한 종목을 빠르게 확인하기 위해 TASS를 사용한다.

사용자는

* 추천 종목
* 추천 이유
* 손절 가격
* 익절 가격
* 보유 기간

만 확인하면 된다.

---

# 11. 성공 기준

TASS가 성공했다고 판단하는 기준

* 추천 종목이 명확하다.
* 추천 이유가 설명 가능하다.
* Rule 기반으로 재현 가능하다.
* 백테스트 결과가 검증된다.
* AI가 동일한 결과를 재현한다.

# 14. 핵심 철학

TASS는 미래를 예측하지 않는다.

TASS는 확률적으로 유리한 종목을 선별한다.

TASS는 감정이 아닌 규칙으로 판단한다.

TASS는 설명 가능한 추천 시스템을 지향한다.

TASS는 백테스트를 통해 지속적으로 발전하는 투자 플랫폼이다.
